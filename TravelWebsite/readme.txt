Walk-Through:
-------------

Created a 2 html files(TravelAgencyHomePage.html, SignInPage.html) and 1 css file(Styling.css).

-In the <head> section of all files, created links to favicon icon and to the css file to add design filters & decorators.

-In the body section:
	* Displayed the whole body in a "flex" format with the flex-direction as column

	* Under the 'header' element:
		> Created a header and added <h1> & <p> elements for displaying the name of the travel website and a small description about it.
		> Displayed the heading for the login page using <h1> tag.

	* Under the 'main' element:
		> Added navigations to other pages of the website usin anchor <a> and <div> tags.
		> Added an image using <img> in the homepage.
		> Added welcome notes using <div> & <p>.
		> Added TABLE element--Displayed highly rated destinations using <div>, <h2>, <table>, <thead>, <tbody>, <tr>, <th>, <td>.
		> Displayed the offers of the day using <h2>, <ul>, <li>.
		> Redirecting to cdc safety measures website using <a> tag and displayed this anchor tag to the right using <float>.

	* Under the footer section:
		> Added "terms and condition" using <a> tags.
		> Added an option to redirect to mail app to send a mail to the Travel agency team using <a> tag and "mailto:".

-In the css file:
	Added various types of css elements to modify the display of webpages. The kind of tags used will be mentioned in the next section.
