Assignment Instructions:

Assignment-1 : Part-B

Create 2 pages website for a coffee shop or Travel or anything with a Usage of a minimum of 5-6 HTML5 tags like <audio>, <video>, <header>, <footer>, etc.

Note: Rich UI is not expected but it should be presentable. Use appropriate color for the website, headings, and layouts, use appropriate images and 
try to make the page broken down into sections for better visual appeal (be creative). Use external CSS file for styling.




Walk-Through:
-------------

Created a 3 html files(homepage.html, menu.html & signin.html) and 1 css file(cssstyles.css).

-In the <head> section of all files, created links to favicon icon and to the css file(cssstyles.css) to add design filters & decorators.

-In the body section:
	* Displayed the whole body in a "flex" format with the flex-direction as column
	* Under the 'header' element:
		> Created a header and added <h1> & <p> elements for displaying the name of the coffee shop and a small description about it.
	* Under the 'main' element:
		> Added navigations to other pages of the website usin anchor <a> and <div> tags.
		> Added video using <video> tag.
		> Added images using <img> tags.
		> Added a breif description about the speciality of the coffee shop using <article>, <h2> & <p> tags.
		> Described different kinds of coffee using <section> tags.
	* Under the section element:
		> Added <h2> and <p> tags inside it to describe each coffee type.
		> Added <audio> element to describe the way the name of the coffee is pronounced.
		> Added <input>s of type "checkbox" and "number" to select a kind of coffee and its quantity respectively.
	* Under the footer section:
		> Added "terms and condition" using <a> tags.
		> Added an option to redirect to mail app to send a mail to the coffee shop team using <a> tag and "mailto:".

-In the css file:
	Added various types of css elements to modify the display of webpages. The kind of tags used will be mentioned in the next section.
